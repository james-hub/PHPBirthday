PHPBirthday ver. 1.3 - August-06-2008
To install phpbirthday follow the below instructon.

1.) upload php files to web server.
2.) Create the MySQL database and import the phpbirthday.sql file.
2.) edit both config.php and main/config.php
3.) edit email.pl and add to crontab to run once a day in the morning.

With some webservers you may need to create a .htaccess file with the below values.

AddType application/x-httpd-php .html
